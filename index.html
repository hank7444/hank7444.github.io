個人簡介

我具有前後端各兩年以上的開發經驗(遠端工作一年半)，在活動網站開發、App 後端 API 設計、電子商務網站建制上均有數年經驗。因為個人對前端技術的熱愛，因此轉換跑道到前端這條路上。

我個性樂觀開朗，樂於分享所知所學，但是對於工作上則會以謹慎的態度來評估需求與開發，注重細節並且實事求是不會過度樂觀。喜歡研究、與同事探討各種新技術、開發方法與流程，並實際導入團隊中。

因為有多年後端開發經驗，所以可以跟設計師、PM、後端工程師很好的協同合作，並且適時提出UX/UI與API規格上的錯誤與改善的建議。

在生涯發展上，我明白走技術這條路只有與時俱進，不斷成長才是正途，所以內心始終是充滿危機感的。因此我常常關注新技術的發展、參與研討會、與不定時採購新書來增進自己的專業能力與知識。

對於產品開發，對我來說更像是養自己的孩子一樣，希望它可以健康卓越的成長。所以我現在研究的重點在如何在協同合作的環境中，開發高度可靠的產品。對我來說，產品品質優良，我的組員能從每一次開發中都能夠學習到新的技術或是方法，大家一起成長卓越，就是我最大的快樂。

碩士

大學畢業後，我順利經推甄管道進入了台大電腦輔助工程組就讀碩士班，並跟隨著康仕仲老師進行人機介面相關領域的學習與研究。在研究所就讀期間，我參與了多個專案與研討會，在老師們細心的指導下，逐漸的增加了簡報、專案製作與團隊協調、合作的能力，並且也學習到老師們做事的態度，對小細節也不偷懶的精神。

對各種事物充滿興趣的我，想要做一些吸引人的東西，因此在跟老師討論論文內容時，我們構想用互動式視覺化的方式來呈現工程資訊。從構想、開發到完成系統，這一路上充滿了困難與挫折，但是看到當時想法，從無到有化為實際的成果時，心中的成就感是難以言喻的，也讓我了解人生不能只做有把握的事情，一定要面對一些挑戰才能有所突破。
