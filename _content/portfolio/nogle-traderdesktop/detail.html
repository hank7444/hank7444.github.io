作品資訊

參考網址：無(內部產品)

作品描述

期貨高頻交易平台，方便交易員將自己的策略上傳後，及時顯示程式自動交易的成果。

製作過程感想

這個案子是我第一次使用 react 全家桶(react + redux + babel + webpack)， 一開始進來公司後端已經有做一部分了，但是問題很多，所以我進去就決定用SPA進行前後端分離並重寫 + 重新切板， 我一直覺得切板是前端工程師必備的技能，一定要能自己掌握，才能很好的用 css 進行 UI 的一些操作。

一開始公司並沒有其他人用 react 進行開發，所以我也硬著頭皮將 react 全家桶的架構建構出來。 最開心的是這個案子讓我有機會用 react 進行實戰，也因此得到許多寶貴的經(踩)驗(雷)。

這邊得到最棒的經驗就是，前端一定要自己弄一個 mockAPI server，這樣就可以只要後端先出 API 文件後， 我們可以先塞假資料進去進行開發，好處是我們可以自己控制各種API的情形，例如延遲或發生錯誤， 而提早針對這些狀況來設計前端元件。
