作品描述

期貨交易監控平台，幫助期貨公司即時監控是否有異常交易情形。

製作過程感想

這案子最印象深刻是需要客製化許多 UI 元件，我從無到有從畫面開始刻並實作功能， 深深覺得刻複雜 UI 並非是一件容易的事，從一開始靜態畫面、資料輸入輸出、多 UI 連動、各種鍵盤與滑鼠事件， 有很多細節需要實作。

再來此專案需要很彈性的列表，可以動態改變要顯示的欄位與顯示欄位的位置。這邊用了 ag-grid, 並且客製化部分功能與 UI， 以符合專案所需。

在歷經多個產品後，很多時候必須有所取捨，隨時要把營運放在第一位，解營運上的 bug 遠遠重要於 解技術上的 bug。 我的經驗是，先有個有彈性的基礎框架，並儘量把各自功能寫得乾淨，即使有時候必須寫很髒的程式碼，但可以讓負面影響只限制 在可控的範圍，等有空擋時間再進行重構，這樣邊可以更早的將有助於營運的功能推出，爭取時間。
